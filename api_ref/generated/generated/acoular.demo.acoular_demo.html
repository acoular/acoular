acoular_demo¶

Demo for Acoular.

To run the demo, execute the following commands:

import acoular

acoular.demo.acoular_demo.run()

Generates a test data set for three sources, analyzes them and generates a map of the three sources.

The simulation generates the sound pressure at 64 microphones that are arrangend in the ‘array64’ geometry, which is part of the package. The sound pressure signals are sampled at 51200 Hz for a duration of 1 second.
