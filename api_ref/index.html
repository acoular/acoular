Reference Manual¶

Modules in Acoular¶

The following modules are part of Acoular:

aiaa

Provides classes for importing AIAA Array Benchmarks.

base

Implements base classes for signal processing blocks in Acoular.

calib

Implements calibration of multichannel time signals.

configuration

Implements global configuration of Acoular.

demo

The Acoular demo.

deprecation

Implements helper functions for deprecation handling.

environments

Implements acoustic environments with and without flow.

fbeamform

Implements beamformers in the frequency domain.

fprocess

Implements blockwise processing methods in the frequency domain.

grids

Implement support for multidimensional grids and integration sectors.

microphones

Implements support for array microphone arrangements.

process

General purpose blockwise processing methods independent of the domain (time or frequency).

sdinput

Input from soundcard hardware using the SoundDevice library.

signals

Implements signal generators for the simulation of acoustic sources.

sources

Measured multichannel data management and simulation of acoustic sources.

spectra

Estimation of power spectra and related tools.

tbeamform

Implements beamformers in the time domain.

tools

Useful tools for Acoular.

tprocess

Implement blockwise processing in the time domain.

trajectory

Implements the definition of trajectories.
