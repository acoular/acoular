Contribution Checklist#

Before submitting a pull request, please make sure that you have done the following:

  • [ ] Read the Contributing section

  • [ ] Ensure your branch is up-to-date with the main branch of the Acoular repository

  • [ ] Verify that the Code Quality Standards standards are fulfilled

  • [ ] Updated the What’s new section in the documentation explaining the changes you made

  • [ ] Append yourself to the author list in the CITATION.cff file
