`https://acoular.org/contributing/ <https://acoular.org/contributing/>`_

* `Documentation <https://acoular.org/contributing/documentation.html>`_
* `Submitting a pull request <https://acoular.org/contributing/pull_request.html>`_

* `Release policy <https://acoular.org/contributing/release.html>`_
* `Deprecation policy <https://acoular.org/contributing/deprecation.html>`_
